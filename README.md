# Final Project

## Author: Andrew Daniel Chihaia 

This is the final project for the Vanier Introduction to Programming.This project implements a student, course, and assignment management
system in Java. It follows object-oriented design principles and includes
unit tests for key methods.

## Features
- Student registration and course management
- Assignment creation and grade calculation
- Automatic ID generation
- Title case formatting using a utility class
- Unit testing with JUnit 5

## How to Run
- Open the project in IntelliJ
- Run the test classes using JUnit
- (Optional) Run Main.java for demo output

## Technologies
- Java
- JUnit 5
